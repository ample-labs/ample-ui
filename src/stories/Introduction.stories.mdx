# ðŸŽ¨ Ample UI (AUI)

Ample UI helps you build awesome React Applications.

We recommend building UIs with a [**component-driven**](https://componentdriven.org) process starting with atomic components and ending with pages.

## Philosophy
1. Every input must be compatible with [**react-hook-form**](https://react-hook-form.com/) library.
2. Must be written in [**Typescript**](https://www.typescriptlang.org/).
3. Inspired by [**MUI**](https://mui.com/).

## How to use

You can browse example stories in the sidebar.
Each components have their own `Story` which contains working examples and codes.
